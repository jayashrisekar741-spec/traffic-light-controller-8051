 # Traffic Light Controller using 8051 Microcontroller

This project implements a basic 3-phase Traffic Light Control system using an 8051 microcontroller. It cycles through Red, Yellow, and Green lights with specific time delays.

## 🚦 Hardware Logic
* **Red LED:** Connected to P1.0
* **Yellow LED:** Connected to P1.1
* **Green LED:** Connected to P1.2
* **Logic:** Active Low (LED turns ON when the pin is 0).

## 🛠️ Software Used
* **Keil uVision 5:** For writing C code and simulation.
* **Language:** Embedded C.

## 📂 Project Structure
* `traffic light 8051.c`: Main source code.
* `traffic light 8051.hex`: Compiled binary file for hardware flashing.
* `traffic light 8051.uvproj`: Keil project file.

## 🚀 How to Run
1. Open the `.uvproj` file in Keil uVision.
2. Build the project to generate the hex file.
3. Use the Debug mode to simulate the I/O port transitions in the Peripheral window
